# From receptor binding to biogeography: multi-scale prediction of filovirus hosts in bats
Gathered here are all of the scripts dedicated to modeling bat NPC1 binding strength to all currently known mammalian filoviruses. 

The scripts include those dedicated to creating the datasets used in our analysis (NPC1DataCollation), the analysis pipeline (NPC1MammalPhyloIterations), summary and exploration of the results (FilovirusResultsMammals), and specific figures.
